An overflow-toggle is collapsed by default to it’s own max-height. The default max-height of an overflow-toggle is 7.5rem (120px). When toggled open, that max-height is overridden to include the whole height of the content, and the state is transitioned from closed to open via CSS.

Attributes

Text rendered into the toggle button.

A boolean attribute describing the state of the overflow-toggle.

Properties

.open

false A boolean property describing the state of the overflow-toggle – true when the overflow-toggle is open, false when it is not.

Parts

The toggle button that, when clicked, shows and hides overflowing content.
