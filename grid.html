bolt.grid.css

Generate a responsive grid for Bolt.

Enter the breakpoints in pixels. You may enter any number of breakpoints separated by spaces. They get converted to media queries in em. For example, '480 960' would generate: 

@media all and (min-width: 30em) {…}
@media all and (min-width: 60em) {…}

Grid columns are created from 1 up to max divisions. For example, if max divisions is 8 or over CSS is generated for:

class="grid-5/8"
The gutter is the fixed space between grid divisions in pixels. It gets converted to rem.

grid-n/d classes set width

grid-left-n/d classes set margin-left

grid-right-n/d classes set margin-right

grid-index-n/d classes set width of child <li>s


Where breakpoint is any number entered in :

breakpoint-grid-n/d classes set width from the given breakpoint

breakpoint-grid-left-n/d classes set margin-left from the given breakpoint

breakpoint-grid-right-n/d classes set margin-right from the given breakpoint

breakpoint-grid-index-n/d classes set width of child <li>s from the given breakpoint
