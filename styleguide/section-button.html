4 #button .button

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Inline-block with padding and centered text ready to be styled as a button. It has no background or border-radius by default. Both the button element and the class button have this base style. The button class can also be used on labels wrapping various form elements.

Example
Link to something
Markup 
<a class="button" href="#something">Link to something</a>
