13.2 #yactions.slide .slide

Toggle example guides Toggle HTML markup

An action class for making slideshows.

Clicks on links pointing to a slide trigger its active state, and deactivate all it's sibling slides. In other words, only one slide amongst siblings can be active at any one time. Slides are also given swipe left and swipe right behaviour.

Note that in this example, the class slide makes the imgs activatable, but the classes slide-block block are needed to define style and transitions.
