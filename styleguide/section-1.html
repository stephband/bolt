.block

block.css:7

For layout. There are two ways a .block can be styled: as a block, for a vertical layout, or as inline-block, for a horizontal layout. block by default, obviously.

Any .block child of a .grid-block becomes an inline-block. Grid layout can be designed by adding .grid-n/d classes to those children.

Applied to a media tag like img, svg, video and canvas, .block will scale the element to fill the available width.

.grid-n/d

grid.scss:195

Sets width to n/d divisions of the horizontal grid. There are 6 divisions available, with a gutter of 1.25rem (20px). Widths are expressed as reduced fractions, so for example there is no .grid-2/4, because that is reduceable to .grid-1/2.

.grid-1/1 - Full width column. .grid-1/2 - Half width column. .grid-1/3 - Third width column. .grid-2/3 - Two thirds width column. .grid-1/4 - Quarter width column. .grid-3/4 - Three-quarters width column. .grid-1/5 - Fifth width column. .grid-... - ... .grid-5/6 - Five-sixths width column.

Parameters:
  • .grid-1/n
    Adaptive width, depending on number of siblings
