2.4 #block.slides-block-2 .slide-block

Toggle example guides Toggle HTML markup

An inline-block set up for left-right transitions between it and it's siblings. slide-blocks are only styled as slides where the js class is on an ancestor, as they require JS to be enhanced as swipeable.
