7.1 #index.crumb-index .crumb-index

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

A '>' character delimited list on one line for making navigation breadcrumbs. lis are inline-blocks, so they should have no spaces between them.

Example
  • path
  • to
  • document
Markup 
<ul class="crumb-index index">
   <li><a href="#path">path</a></li
  ><li><a href="#to">to</a></li
  ><li><a href="#document">document</a></li>
</ul>
