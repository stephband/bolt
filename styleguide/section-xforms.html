12.2 #xForms.2 .radio-label

A radio input wrapped in a label that gives it custom style. Depends on JS, so the style is enhanced only where the class js is present on a parent.

Example
Markup 
<label class="radio-label" for="radio-1">
  <input type="radio" id="radio-1" />
  Yes, choose me!
</label>

File inputs can be wrapped in buttons for custom styling. The caveat: there is no way to display the selected file name if you do this.

Example
Markup

  Select file...
