13 #yactions Actions

Action classes progressively enhance elements with behaviours using JavaScript (found in bolt-musees.js). Actions are instantiated lazily when they are first activated, so there is no need to initialise them by calling any jQuery plugins.

When a link is clicked and its href references a node in the DOM that has an action class, an "activate" event is triggered on that node, and it is given the class "active". All links that point to it are given the class "on". The type of behaviour determines how the node can be deactivated.

The activate and deactivate events are jQuery special events. Nodes with action classes can be activated via JS, simply by triggering an activate event: 

jQuery('#my-toggle').trigger('activate');

Parameters:
  • .active
    applied to elements with action classes
  • .on
    applied to as with an href that points to an element with an action class (or labels with a for attribute pointing to a radio or checkbox).

13.2 #yactions.slide .slide

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

An action class for making slideshows.

Clicks on links pointing to a slide trigger its active state, and deactivate all it's sibling slides. In other words, only one slide amongst siblings can be active at any one time. Slides are also given swipe left and swipe right behaviour.

Note that in this example, the class slide makes the imgs activatable, but the classes slide-block block are needed to define style and transitions.

Example

Markup 
<div class="slides-block block">
  <img class="slide-block block slide active" id="slide-1" src="//source.unsplash.com/category/nature/520x310"
  /><img class="slide-block block slide" id="slide-2" src="//source.unsplash.com/category/technology/520x310"
  /><img class="slide-block block slide" id="slide-3" src="//source.unsplash.com/category/food/520x310" />
</div>
<a href="#slide-1">1</a>
<a href="#slide-2">2</a>
<a href="#slide-3">3</a>

13.3 #yactions.toggle .toggle

An action class for creating drawers, submenus or anything else that has open/closed state.

Clicks on links pointing to a .toggle toggle its active state. Note that in this example, the class toggle gives the p active/inactive behaviour, but the classes toggle-block block are needed to give it style and transitions.
