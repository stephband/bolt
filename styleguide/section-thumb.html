10 #thumb .thumb

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

Base class for thumbnails. A .thumb is an icon or avatar or other image. A .thumb hides text content by pushing it outside the hidden overflow, displaying only it's background-image. The user cannot interact with a background-image in the way they can with an <img/>). This should be the main consideration when deciding to use a .thumb or an <img/>. Thumb is in essence an image replacement technique.

A default thumb is a square of 64x64px. Add an image by defining a background-image. The image is centered and 'cropped' via background-size and background-position.

Example
Thumb text
Markup 
<a class="thumb" href="#" style="background-image: url('//source.unsplash.com/category/nature/128x80')">Thumb text</a>
