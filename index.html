For layout. There are two ways a .block can be styled: as a block, for a vertical layout, or as inline-block, for a horizontal layout. block by default, obviously.

Applied to a media tag like img, svg, video and canvas, .block will scale the element to fill the available width.

Thumbs.

.thumb

Thumb text

Base class for thumbnails. A .thumb is an icon or avatar or other image. A .thumb hides text content by pushing it outside the hidden overflow, displaying only it’s background-image. The user cannot interact with a background-image in the way they can with an &lt;img/&gt;). This should be the main consideration when deciding to use a .thumb or an &lt;img/&gt;. Thumb is in essence an image replacement technique.

A default thumb is a square of 64x64px. Add an image by defining a background-image. The image is centered and ‘cropped’ via background-size and background-position.

<a class="thumb" href="#" style="background-image: url('//source.unsplash.com/category/nature/128x80')">Thumb text</a>
